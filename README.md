# linux-console
Script para interação com a porta serial 

1º Dê a permisão de execução para o script -> sudo chmod +x serial
2º Chame o script ./serial /dev/ttyUSB0 9600
                      |         |         | 
                      |         |         |-> Velocidade  
                      |         |-> Caminho para a porta serial
                      |-> Script
